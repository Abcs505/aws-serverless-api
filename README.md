# ⚡ AWS Serverless REST API

## 📌 Project Overview
A serverless REST API built with AWS Lambda 
and API Gateway using Python.

## 🌐 Live API URL
[Click here to test the API](https://t0vu07wmt8.execute-api.ap-south-1.amazonaws.com/production/hello)

## ☁️ AWS Services Used
- **AWS Lambda** — Serverless function
- **API Gateway** — REST API endpoint
- **Python 3.14** — Programming language
- **CloudWatch** — Monitoring and logs

## 🚀 How It Works

User hits URL → API Gateway →
Lambda runs → Response returned!
